working in a smart web application designed to 'bridge the gap between job seekers and employers' by analyzing resumes (CVs) against job descriptions using AI.

It provides deep insights, comparison metrics, and communication evaluation to help users improve their chances in the job market.

 🔍 Features

 📄 Resume vs Job Description Analysis

* Compares CVs with Job Descriptions using AI
* Highlights:
  * Skill gaps
  * Matching percentage
  * Missing keywords
* Uses *Natural Language Models (NLM)* for intelligent comparison

 🧠 Communication Skill Tester

* Evaluates candidates through:
  * ✍️ Written tests
  * 🎤 Voice-based tests
* Analyzes:

  * Grammar
  * Clarity
  * Confidence
  * Communication effectiveness

Cross-Check & Data Analysis

* Performs multiple validation checks on:
* Resume content
* Input data
* Ensures accuracy and reliability of results

 🛠️ Tech Stack

* Backend: Python
* AI Models:
  1. CV vs JD Comparison Model (NLM-based)
  2. Communication Evaluation Model (Voice + Text)
* Web Integration: Full-stack web application
